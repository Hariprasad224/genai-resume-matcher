# GenAI Resume–JD Matcher

This application uses:
- Gemini embeddings
- TF-IDF similarity
- FastAPI backend
- Streamlit frontend

## Features
- Resume upload (PDF)
- Job description analysis
- Match percentage
- AI-powered feedback

## Tech Stack
FastAPI | Streamlit | Gemini | TF-IDF

## How to Run
1. Add Gemini API key in `.env`
2. Run backend
3. Run frontend
